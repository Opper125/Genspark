အသုံးပြုပုံညွှန်ကြားချက်များ

  • အနည်းဆုံး API key တစ်ခု ထည့်သွင်းပေးရပါမယ်
  • Google Gemini နှင့် Groq တို့က လုံးဝ အခမဲ့ဖြစ်ပြီး unlimited အသုံးပြုနိုင်ပါတယ်

  • API keys များကို browser ရဲ့ localStorage မှာသာ သိမ်းဆည်းပါတယ် (လုံခြုံပါတယ်)
  • ဘယ် server မှာမှ သင့် API keys တွေ မသိမ်းပါဘူး
